Design Decisions

Here I will discuss which and why I used which HTML Element.

The main Idea

The main idea of my Website is to look minimalistic, modern with a touch of Computer Science and Retro Gaming.

This is done by using a more Code like looking font and a VHS Screen video that acts as the background to add this old VHS noise and feeling

The Design Elements

Structural HTML Elements

<div class="content">
I used the <div class="content"> to seperate the video, which functions as the background, and the contents. This will insure that when I can apply CSS rules on the content more easyaly.
<header>
The <header> is used to describe the Navigation Menu at the top of the website. It is fixed at the top of the Website.
<nav>
The <nav> hold the <ul> with each <li> holding a link to a site. The <nav> is used because it is the smantic tag excectly used for this propuse.
<main>
<main> is used to describe the main contents of the site. It has an ID=top because I wanted the functionality that If you are at the bottom of the site, you could easaly get to the top again. This is done by creating an href that links to the #top of the site.
<articel>
The article tag is used to define the main article. Sometimes I used a <aside> which does not have to be in main article.
<section>
A <section> is used to structure an article in itself. Here I used other Elements like <h1>, <ul>, <p> or <img>.
<footer>
Similar to the <header>. It was used to create a footer that showed the "back to top link" as well some other links you usally find at the footer.
<address>
The Address was used to surprise write about my address. I used the <br> there to make the formatting better (as learned in the Lecture).
