Game of Not

Introductoin

This game is about doing something NOT.

Wait what?

Yeah. You read it correclty. This game wants to help you to participate in helping the planet. This is done by asking you to do something not. We know, that following those kind of rules are not easy all the time. But we want to reward you for participating if you can and encourage long term participartation.

How to play

Playing is easy: if you do something that's on the list below, you get the corresponding points. The points will be collected in a printed table (table.pdf) and will summed up at the end of the week. The goal is to get every week a better highscore.

For example, if you have trash in your hands, don't throw it in the nature and instead use a bin which will give you 10 Points. Or if you are able to not use the car for short routes, that will give you 5.

The best way to play this game is with your friends and family, so you can all engage in saving the planet while competing with the others for the most points. Having this competetion will engage all of you to make the difference.

A (printable) template for adding the points per week can be found here: table.pdf. But the best would be if you would write it in a document right away, to keep track with it.

What can you do with the points?

The points are an indicator for how much you think before you might do something bad to the environment. The goal is to get more and more points per week. To keep track, you can download our PDF.
